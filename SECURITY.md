# Security Policy

## Supported Versions

| SlateTerm version     | Supported? |
| --------------------- | ---------- |
| `1.x.x`               | &check;    |
| Slate                 | &cross;    |

> [!IMPORTANT]
>
> Support for Slate, SlateTerm's older version, has officially dropped. You can still [view the source code](https://github.com/triflare/slate).

## Reporting a Security Vulnerability

To report a security vulnerability with SlateTerm, use GitHub's native feature OR send an e-mail to `triflare.dev@proton.me`. We aim to acknowledge reports within 72 hours and provide a full response within 90 days.
